# v3.2.1

本次更新将软件入口改为原生桌面 UI，双击 `tdl.exe` 会直接打开控制台窗口。

## 更新内容

- 新增 Tkinter/ttk 原生桌面控制台，不再需要先打开浏览器页面。
- 软件启动准备过程会直接显示在 UI 中，包括读取配置、启动 Telegram 客户端、启动机器人、扫描任务等日志。
- 新增实时下载表格：显示任务、会话、消息 ID、文件名、进度、大小、速度、状态和保存路径。
- 新增机器人/任务面板：显示 config 任务和机器人命令任务的完成数、失败数、待恢复数量和最近机器人状态消息。
- 新增原生配置页：可交互修改保存路径、并发参数、超时参数、日志等级和 Clash 自动切换配置。
- 启用 Windows 高 DPI 适配，改善高分屏下文字和控件缩放效果。
- 打包入口切换为 `gui_launcher.py`，发布的 exe 使用无控制台窗口模式。
- 发布脚本加入原生 UI 编译和非窗口逻辑测试。

## 注意

- `max_download_task` 这类影响工作线程数量的配置，保存后建议重启软件完全生效。
- 原来的 Web 控制台仍保留，可作为备用远程查看入口。
