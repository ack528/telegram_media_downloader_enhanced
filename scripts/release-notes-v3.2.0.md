# v3.2.0

本次更新加入全新的现代化 Web 控制台，方便长时间运行时直接查看下载、机器人和配置状态。

## 更新内容

- 重做 Web 首页 UI：新增总览、文件下载、机器人、配置四个交互面板。
- 实时显示各文件下载信息：文件名、任务 ID、消息 ID、进度、速度、已下载大小和保存路径。
- 新增任务状态面板：显示 config / 机器人任务来源、完成数、失败数、待恢复数量和任务进度。
- 新增机器人交互面板：显示机器人是否启用、机器人任务和最近一次状态消息。
- 新增可交互配置页面：支持修改常用配置、Clash 自动切换参数、media_types、file_formats 和 chat 配置。
- 新增 `/api/dashboard` 与 `/api/config` 接口，为后续 UI 扩展打基础。
- 增加 Web API 和首页渲染测试，减少后续更新时界面接口损坏风险。

## 注意

- `max_download_task`、`web_host`、`web_port` 等运行期结构相关配置保存后建议重启软件生效。
- API 密钥和 bot token 不会在页面中明文展示；Clash 密钥可在配置页按需填写保存。
